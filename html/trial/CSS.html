CSS dolgok

CSS használata - 16 pont

  • Legalább 1 CSS fájl használva - -1p?
  • Legalább 1 tag (elem) szelektor - 1p
  • Legalább 1 id szelektor (#valami) - 1p
  • Legalább 1 class szelektor (.valami) -1p
  • Legalább 1 attribútum szelektor (valami[attr] vagy valami[attr="val"]- 1p
  • Legalább 1 kapcsolat szelektor (space, >, ~, +) - 1p
  • ---
  • Z-index - 1p
  • "Úsztatás és annak megszüntetése (float és clear)" -1/2p
  • Pseudo-elemek használata (before, after, selection, first-letter, first-line, stb.) - 1/2p
  • van 1 igényes nyomtatási "stíluslap" 1 oldalhoz - 2p
  • Minden HTML oldalhoz van igényesen elkészített nyomtatási stíluslap - 3p

HTML elemek testreszabása - 10 pont

  • border használata (szegély) (0/1 pont)
  • Lekerekített sarkok használata (border-radius) (0/1 pont)
  • Háttérszín / háttérkép használata (0/1 pont)
  • Szövegek formázása (letter-spacing, line-height, text-align, text-decoration) (0/1 pont)
  • Pozicionálás (absolute, fixed, relative, static, sticky) (mindegyik 1 pont, max 3 pont)
  • Árnyékok használata (HTML elemekre vagy szövegekre) (0/1 pont)
  • Transzformálás használata (eltolás, forgatás, skálázás, nyírás)(mindegyik 1 pont, max 2 pont)
